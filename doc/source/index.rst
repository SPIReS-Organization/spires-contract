spires-contract
===============

Executable ``xarray`` boundary validators for the SPIReS package family — the
schemas (dims, dtypes, coords) every stage validates against where data crosses
a package boundary.

This package is part of the `SPIReS family <https://spires.readthedocs.io/>`_.
